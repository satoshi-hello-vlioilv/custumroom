# custumroom — ルームプランナー 3D

ブラウザで動く 3D ルームプランニングアプリです。単一の HTML ファイル（`index.html`）で完結しており、インストール不要で利用できます。

## デモ

`index.html` をブラウザで直接開くだけで動作します。サーバー不要。

## 主な機能

### 3D ビュー
- **Three.js r160** による PBR レンダリング（物理ベースのマテリアル）
- スマートカットアウェイモード：外壁を自動で透過表示
- 軌道カメラ操作（ドラッグ・ピンチで回転・ズーム）

### 2D フロアプランエディタ
- グリッドスナップ（0.5m 単位）で壁と床セルが完全一致
- 壁の描画・削除、床の塗りつぶし
- 開口部（ドア・窓・ガラス戸）の配置
- 3D ビューとリアルタイム同期

## 使い方

### 基本操作

| 操作 | 説明 |
|------|------|
| 左ドラッグ | カメラ回転 |
| 右ドラッグ / 中ドラッグ | カメラパン |
| スクロール | ズーム |
| 家具クリック | 選択（プロパティパネルが開く） |

### 壁モード

ツールバーの「壁モード」をオンにして左ドラッグすると壁を描画できます。  
2D エディタ（「2D 編集」ボタン）では壁タイプ・開口部タイプを選択して詳細編集が可能です。

### 床ペイントモード

「床ペイント」をオンにして床材を選択後、左ドラッグで床を塗り、右ドラッグで消せます。

### 保存・読み込み

- 「保存」ボタン：JSON ファイルをダウンロード
- 「読み込み」ボタン：保存した JSON を読み込む

## 開発

```bash
# リポジトリをクローン
git clone https://github.com/satoshi-hello-vlioilv/custumroom.git
cd custumroom

# index.html をブラウザで開く（サーバー不要）
open index.html
```

ライブリロードが必要な場合は任意のローカルサーバーを使用してください。

```bash
# Python の場合
python3 -m http.server 8080
```
